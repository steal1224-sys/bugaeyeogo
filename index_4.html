🔧 Firebase 연동 안내

Firebase를 연동하면 모든 기기·브라우저에서 서평이 공유됩니다 (무료)

1
Firebase 콘솔 접속
console.firebase.google.com에서 구글 계정으로 로그인 후 프로젝트 추가를 클릭하세요.
2
웹 앱 등록
프로젝트 홈 → </> (웹) 아이콘 클릭 → 앱 닉네임 입력 후 등록 → 화면에 표시된 firebaseConfig 값을 복사해 두세요.
3
Firestore 데이터베이스 생성
왼쪽 메뉴 → Firestore Database → 데이터베이스 만들기 → 테스트 모드로 시작 (나중에 규칙 수정 가능).
4
HTML 파일에 설정값 붙여넣기
이 HTML 파일을 메모장(또는 에디터)으로 열어 아래 부분을 찾아 수정하세요.
// ⚙️ Firebase 설정 — 여기에 복사한 값을 붙여넣으세요 const FIREBASE_CONFIG = { apiKey: "여기에 붙여넣기", authDomain: "your-project.firebaseapp.com", projectId: "your-project-id", storageBucket: "your-project.appspot.com", messagingSenderId: "000000000000", appId: "1:000:web:0000" };
5
파일 저장 후 브라우저에서 다시 열기
설정이 완료되면 데모 모드 배너가 사라지고 Firebase에 서평이 저장됩니다.
